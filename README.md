# School

Адаптивный проект на `React + Vite + Tailwind CSS`.

## Стек

- React
- Vite
- Tailwind CSS

## Запуск

### Windows

1. Установите Node.js LTS: https://nodejs.org/en/download
2. Откройте PowerShell от имени пользователя и разрешите запуск локальных скриптов:

```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```

3. Если PowerShell запросит подтверждение, введите `A` или `Y`.

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
